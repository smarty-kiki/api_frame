# Demo  
示例实体






### Demo列表  
----
**功能：**通过多个筛选条件来查询Demo列表  
**请求方式：**`GET`  
**请求地址：**  
```
/demos  
```
**参数：**  

|参数键名|类型|必传|描述|
|----|----|----|----|
|name|string|可选|通过名称筛选|
|note|string|可选|通过备注筛选|

**返回值：**  
```json
{
    "code": 0,
    "msg": "",
    "data": [
        "count": 0, // 查询出来的Demo数量
        "demos": [] // 查询出来的Demo数组
    ],
}
```
Demo数组包含多个Demo结构，单个结构的格式为
```json
{
    "id": 0, // 主键
    "name": "string", // 名称 
    "note": "string", // 备注 
    "create_time": "2026-02-09 17:58:34", // 创建时间
    "update_time": "2026-02-09 17:58:34", // 最后一次修改时间
}
```





### 添加Demo  
----
**功能：**添加Demo  
**请求方式：**`POST`  
**请求地址：**  
```
/demos/add  
```

**参数：**  

|参数键名|类型|必传|描述|
|----|----|----|----|
|name|string|必传|名称|
|note|string|可选|备注|

**POST请求体：**  
```json
{"name":"名称","note":"备注"}  
```

**返回值：**  
```json
{
    "code": 0,
    "msg": "",
    "data": [
        "id": 1 //Demo id
        "name": "",
        "note": "",
        "create_time": "2021-01-01 00:00:00",
        "update_time": "2021-01-01 00:00:00"
    ]
}
```



### 查询Demo   
----
**功能：**查询单个Demo信息  
**请求方式：**`GET`  
**请求地址：**  
```
/demos/detail/{{demo_id}}
```

**`URL`中的变量：**

|变量键名|类型|必传|描述|
|----|----|----|----|
|demo_id|id|必传|Demo的主键，`id`|

**参数：**  

|参数键名|类型|必传|描述|
|----|----|----|----|
|name|string|可选|通过名称筛选|
|note|string|可选|通过备注筛选|

**返回值：**  
```json
{
    "code": 0,
    "msg": "",
    "data": [
        "id": 1 //Demo id
        "name": "",
        "note": "",
        "create_time": "2021-01-01 00:00:00",
        "update_time": "2021-01-01 00:00:00"
    ]
}
```
