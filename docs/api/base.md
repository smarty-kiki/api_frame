# 基础接口  
框架中自带的一些接口




### hello world  
----
**功能：**向世界打声招呼  
**请求方式：**`GET`  
**请求地址：**  
```
/
```

**返回值：**  
```
hello world
```







### 健康检查
----
**功能：**响应健康检查请求  
**请求方式：**`GET`  
**请求地址：**  
```
/health_check
```

**返回值：**  
```
ok
```







### 错误码字典 
----
**功能：**获取错误码字典  
**请求方式：**`GET`  
**请求地址：**  
```
/error_code_maps  
```
**返回值：**  
```json
{
    "DEMO_NOT_FOUND": "无效的 demo",
    "DEMO_REQUIRE_NAME": "未传入 name"
}
```
